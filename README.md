
# FizzBuzz

Using the django framework & db.sqlite3, we're able to create object, display list and retrive detail objects

## Installation

Download

```bash
  git clone https://github.com/RedericYoung/django-fizzbuzz.git
```

Navigate to folder

```bash
  cd fizzbuzz
```
    
Run Server

```bash
  python manage.py runserver
```
